A plugin to replace the default Dokan dashboard with a fresh, modern, and user-friendly design.

== Description ==

Tired of the default Dokan dashboard? Awesome Dokan completely overhauls the vendor dashboard experience with a clean, modern, and intuitive interface. This plugin replaces the default template with a new one focused on better data visualization and a more professional look and feel.

== Installation ==

1. Upload the `awesome-dokan` folder to the `/wp-content/plugins/` directory.
2. Activate the plugin through the 'Plugins' menu in WordPress.
3. Make sure you have the Dokan Lite plugin installed and activated.
4. Visit your vendor dashboard to see the new design!
